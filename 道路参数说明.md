# 道路类型通行时间参数说明

## 📊 道路分类与通行时间设置

本系统定义了5种不同类型的道路，每种道路根据实际交通特性设置了不同的**速度因子(speed_factor)** 和 **拥堵概率(congestion_prob)**：

### 1. 主干道 (Main Road) - 🚗 高速通行

- **速度因子**: 1.2
- **拥堵概率**: 0.2 (20%)
- **通行特点**: 
  - 宽阔多车道，适合高速通行
  - 通行时间为基准长度 × 0.83 (通行更快)
- **实际场景**: 城市主干道、环城公路、高速连接线

### 2. 商业步行街 (Commercial Pedestrian Street) - 🚶 人流密集

- **速度因子**: 0.5
- **拥堵概率**: 0.4 (40%)
- **通行特点**: 
  - 行人众多，机动车限速
  - 通行时间为基准长度 × 2.0 (通行较慢)
- **实际场景**: 购物中心步行街、商业区街道、夜市街道

### 3. 学校周边 (School Zone) - 🏫 时段限速

- **速度因子**: 0.6
- **拥堵概率**: 0.5 (50%)
- **通行特点**: 
  - 上下学时段学生众多，限速通行
  - 通行时间为基准长度 × 1.67 (通行较慢)
  - 时间窗口限制在540-1020分钟(9:00-17:00)
- **实际场景**: 中小学门口道路、大学校园周边

### 4. 居民区 (Residential Area) - 🏘️ 限速通行

- **速度因子**: 0.7
- **拥堵概率**: 0.3 (30%)
- **通行特点**: 
  - 道路狭窄，需减速慢行
  - 通行时间为基准长度 × 1.43 (通行略慢)
- **实际场景**: 小区内部道路、社区街道

### 5. 施工路段 (Construction Zone) - 🚧 通行受阻

- **速度因子**: 0.3
- **拥堵概率**: 0.8 (80%)
- **通行特点**: 
  - 道路被施工占用，车道减少
  - 通行时间为基准长度 × 3.33 (通行极慢)
  - 极高概率发生拥堵
- **实际场景**: 道路维修、地铁施工、管道改造

## 📐 通行时间计算公式

```
实际通行时间 = 基准道路长度 × (1 / 速度因子)

其中：
- 基准道路长度: 由 segment_length_mean 和 segment_length_std 控制
  (默认: 平均80米，标准差20米)

示例计算：
- 主干道: 80米 × (1/1.2) = 66.7米通行时间 → 通行最快
- 商业步行街: 80米 × (1/0.5) = 160米通行时间 → 通行较慢
- 施工路段: 80米 × (1/0.3) = 266.7米通行时间 → 通行极慢
```

## 🔧 参数调整建议

如果你想调整各类型道路的通行特性，可以修改 `_rand_road_type` 方法中的参数：

```python
def _rand_road_type(self, j, cj):
    scenarios = [
        {
            'type': 0,
            'scenario': '道路名称',
            'time_window': {'E': 开始时间, 'L': 结束时间},  # 分钟
            'color': '颜色代码',
            'speed_factor': 速度因子,  # 越大通行越快
            'congestion_prob': 拥堵概率,  # 0-1之间，越大越容易拥堵
        },
        # ... 其他道路类型
    ]
```

## 📈 实际效果

根据最新运行结果：

| 道路类型 | 速度因子 | 相对通行时间 | 拥堵概率 |
|---------|---------|------------|---------|
| 主干道 | 1.2 | 0.83x | 20% |
| 居民区 | 0.7 | 1.43x | 30% |
| 学校周边 | 0.6 | 1.67x | 50% |
| 商业步行街 | 0.5 | 2.00x | 40% |
| 施工路段 | 0.3 | 3.33x | 80% |

这样的设置更符合实际交通情况：
- 算法会优先选择主干道通行
- 施工路段和商业步行街被视为需要绕开的高成本路径
- 学校周边在高峰时段有更高拥堵概率

## 💡 优化策略建议

基于新的通行时间设置，算法会自动优化：
1. **优先主干道**: 主干道通行时间最短，应优先选择
2. **绕行施工路段**: 施工路段成本极高，算法会主动绕开
3. **时段感知**: 学校周边在时间窗口外可能享有更低成本
4. **拥堵预防**: 高拥堵概率道路会被视为风险路径
