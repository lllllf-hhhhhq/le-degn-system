# LE-DEGN 实验数据设计方案

## 1. 实验目标

本实验旨在全面评估LE-DEGN（语言增强动态边中心图网络）在单双向车道混杂街区街景拍摄车路径优化任务中的性能表现。实验设计遵循国际期刊论文标准，重点验证以下核心假设：

- LE-DEGN在混合路网环境中显著优于传统基线方法
- 各组件（ERFM、SA-DGWN、LHH）协同作用产生正向效果
- 系统在不同规模路网和拥堵场景下具有鲁棒性

## 2. 实验设计

### 2.1 基准实验

#### 实验A：LE-DEGN vs 基线方法对比
- **实验组**: LE-DEGN (ERFM + SA-DGWN + LHH)
- **对照组**: DHAN + Nearest Neighbor
- **评估指标**: AOCC、最终成本、收敛速度、逃逸次数
- **统计方法**: 10次独立实验取平均值，报告标准差

#### 实验B：可扩展性分析
- **自变量**: 路网规模（节点数: 40, 60, 80, 100）
- **评估指标**: AOCC、执行时间、内存占用
- **预期结果**: LE-DEGN保持稳定性能，基线方法性能下降

#### 实验C：拥堵鲁棒性测试
- **自变量**: 拥堵道路比例（10%, 20%, 30%）
- **评估指标**: AOCC、路径重规划次数、总成本变化
- **预期结果**: LE-DEGN在重度拥堵下保持优势

### 2.2 组件贡献度分析

#### 实验D：组件消融研究
| 组件配置 | ERFM | SA-DGWN | LHH |
|---------|------|---------|-----|
| Full LE-DEGN | ✓ | ✓ | ✓ |
| ERFM-only | ✓ | ✗ | ✗ |
| ERFM+LHH | ✓ | ✗ | ✓ |
| ERFM+SA-DGWN | ✓ | ✓ | ✗ |

#### 实验E：LHH启发式模板性能分析
- **评估指标**: 各模板被选中频率、平均逃离成本、偏好权重
- **分析内容**: greedy_widest_road、min_penalty_escape、max_connectivity_escape、adaptive_astar_escape、multi_hop_greedy_escape

### 2.3 道路类型影响分析

#### 实验F：道路类型分布对性能的影响
- **道路类型**: 商业步行街、学校周边、主干道、居民区、施工路段
- **评估指标**: 各类型道路覆盖率、平均通行时间、拥堵发生率
- **参数控制**: speed_factor、congestion_prob分布

## 3. 评估指标

### 3.1 主要指标
- **AOCC (Area Over the Convergence Curve)**: 值越高越好，表示算法在时间预算内的整体性能
- **最终总成本 (Total Final Cost)**: 服务代价 + 转移代价，越低越好
- **收敛速度**: 达到最优解90%所需时间

### 3.2 次要指标
- **逃逸次数 (Escape Count)**: LHH引擎触发次数
- **重规划次数**: ERFM重新规划次数
- **路径质量**: 初始成本 vs 最终成本改善率

- LHH模板选择频率和成本

## 7. 预期实验结果

基于代码实现和算法设计，预期实验将展示：

1. **性能优势**: LE-DEGN的AOCC指标相比基线提升15-30%
2. **鲁棒性**: 在30%拥堵场景下仍保持稳定性能
3. **可扩展性**: 节点数从40增至100时，性能下降<10%
4. **组件协同**: Full LE-DEGN相比任一组件缺失版本均有显著提升
5. **LHH贡献**: 在高拥堵场景下，LHH组件贡献度>40%
